# Julesfress.github.io

Site perso hébergé sur GitHub Pages. La page d'accueil (`index.html`) liste mes
projets sous forme de cartes ; chaque projet vit dans son propre dossier
(ex. `Sondpres/`).

## Ajouter un nouveau projet

1. Créer un dossier à la racine pour le projet (code, données, image de rendu, etc.).
2. Dans `index.html`, dupliquer un bloc `<article class="card">...</article>`
   à l'intérieur de `<div class="grid">`, et l'adapter (image, tag, titre,
   description, lien).
3. Le style (`assets/css/style.css`) s'occupe de la mise en page — pas besoin
   d'y toucher sauf changement de design.
